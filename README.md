# Terminal Portfolio

This project is a portfolio website modeled after a terminal interface, offering a unique way to navigate through sections like "Home", "Portfolio", "Skills", "Education", "Art", and "Contact". This innovative approach provides an engaging user experience for visitors, showcasing projects and skills in a format reminiscent of a computer terminal.

## Features

-   **Terminal-like Interface**: Interact with the portfolio through a command-line styled interface.
-   **Dynamic Navigation**: Use directory-style navigation to explore different pages like a file system.
-   **Animated Transitions**: Experience smooth and visually appealing transitions as if scripts are being executed in a terminal.

## Installation

To set up this project locally, follow these steps:

```bash
git clone https://github.com/Iain-Crowe/terminal-portfolio.git
cd terminal-portfolio
npm install
```

## Technologies Used

-   React.js
-   TailwindCSS
